# hackathonProject

This is our hackathon project, a wordle-like game called Hello Worldle where you try to guess the programming language and are given clues after each guess.

How to Play: Simply input the name of a programming language (case-insensitive) and you'll receive clues like whether the language was released before, or after the one you guessed, and if you're close (within 5 years). Then keep guessing until you get it right!
